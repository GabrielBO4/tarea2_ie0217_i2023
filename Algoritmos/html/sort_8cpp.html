void quickSort ( int  arr[],

)

Implementación del algoritmo de ordenamiento Insertion Sort.

Prototipo de la función del algoritmo de ordenamiento Quick Sort.

Parameters
arrArray que se va a ordenar
lowÍndice más bajo del arreglo
highÍndice más bajo del arreglo Elige un elemento pivote del arreglo y particionar el arreglo alrededor de ese elemento. Todos los elementos menores que el pivote se colocan a la izquierda de este, y todos los elementos mayores se colocan a la derecha. Luego, el proceso se repite recursivamente para los sub-arreglos antes y después del pivote hasta que todo el arreglo esté ordenado.
